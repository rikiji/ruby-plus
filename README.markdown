ruby-plus
=========

A couple of enhancments to ruby standard library i often use in my projects.

* `file.rb`  
  Methods useful to parse files faster (csv and more)
* `string.rb`  
  Japanese parsing and romanization of kana
* `array.rb`  
  Random array utilities
